# Invart PNG Brand Assets

The current public brand kit is PNG-only and derived from the selected original Invart logo image. Do not use the earlier SVG reconstruction attempts for release materials; their geometry and texture do not match the approved PNG direction.

## Canonical Directory

Use `png-from-original/` for public docs, release assets, and project previews.

## Recommended Assets

- `png-from-original/invart-logo-original-master-1774x887.png`: preserved source master.
- `png-from-original/invart-logo-horizontal-1600x800.png`: README and general documentation.
- `png-from-original/invart-logo-docs-header-1200x400.png`: docs header and compact website placement.
- `png-from-original/invart-logo-og-1600x900.png`: social preview and release announcement image.
- `png-from-original/invart-logo-square-preview-1024x1024.png`: square preview cards.
- `png-from-original/invart-mark-from-original-512x512.png`: app icon / large favicon source.
- `png-from-original/invart-mark-from-original-64x64.png`: browser favicon.

## Notes

- These are raster assets and intentionally preserve the original generated logo's soft edges, color texture, and letterform feel.
- SVG/vector recreation is deferred until the mark and wordmark can be redrawn to match this PNG style closely.
