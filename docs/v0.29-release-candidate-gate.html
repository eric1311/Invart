v0.29 1.0 Release Candidate Gate

A local pre-1.0 gate for tests, docs, roadmap, benchmarks, demo artifacts, and evidence integrity.

Goal

Provide one local command that proves the control-plane loop is complete enough for 1.0 RC review.

Gate Flow

pytest->docs->roadmap->benchmarks->pre-v1 demo->evidence verify->RC report

Reports

The gate emits release-candidate-report.json and release-candidate-report.html. The HTML report is intended for quick human review; the JSON report is intended for CI and automation.

CLI

kappaski release-candidate verify --out-dir .kappaski/rc
kappaski rc verify --out-dir .kappaski/rc
kappaski rc verify --out-dir .kappaski/rc --skip-pytest

Tests

PYTHONPATH=src /usr/bin/python3 -m pytest -q tests/test_core.py::test_v029_release_candidate_gate_requires_complete_product_artifacts
PYTHONPATH=src python3 -m kappaski.cli eval benchmark --suite v0.29-release-candidate-gate

Boundary

v0.29 is local 1.0 RC readiness. It is not hosted enterprise GA, IdP integration, kernel-level enforcement, or a full SIEM exporter.
