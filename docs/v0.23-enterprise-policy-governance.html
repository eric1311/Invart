v0.23 Enterprise Policy Governance

Profiles become governed objects with registry, pinning, verification, and raw-content display policy.

Governance Objects

ObjectPurpose
ProfileRegistryOwner, scope, version, hash, profile payload
ProfilePinDistributed immutable-ish bundle for repo/team/session
Raw content policyvisible, folded, truncated, redacted, hidden
Break-glass reviewLedger-backed exception review readiness

CLI

kappaski profile registry --owner security --profile '{"scope":"repo","name":"repo","mode":"managed"}'
kappaski profile pin --registry registry.json --scope repo --name repo --distributed-by security --out pin.json
kappaski profile verify --registry registry.json --bundle pin.json
