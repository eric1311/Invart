v0.8 LLM Reviewer Provider

OpenAI-compatible semantic reviewer provider, raw-content evidence handling, LLM deny explanations, and deterministic critical-rule protection.

Implemented Scope

  • OpenAI-compatible provider via environment configuration.
  • LLM reviewer may recommend deny with structured proof/audit explanation.
  • raw_content can enter review evidence but is redacted, folded by default, summarized, and truncated for display.
  • Provider failure remains governed by review mode and policy fallback.
