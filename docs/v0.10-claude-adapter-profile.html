v0.10 Claude Code Adapter Hardening

Adapter profile and environment surface for the first hardened Claude Code integration.

Implemented Scope

  • Claude Code is the first hardened adapter target.
  • Environment keys are recorded by default.
  • Secret-like values are redacted; non-secret values are folded/truncated.
  • Child process supervision targets strong consistency and records degraded subprocess mode explicitly.
  • Local Claude Code wrapper/hook bridge ingests hook JSONL events, runs the child command, writes ledger/proof artifacts, and records process supervision metadata.

Verification

kappaski adapter profile --kind claude-code
kappaski adapter claude-code-check --binary claude
kappaski adapter claude-code --hook-events hooks.jsonl --out-dir artifacts -- python3 -c "pass"
kappaski adapter claude-code --target . --out-dir /tmp/kappaski-claude-real --session-id ks_real_claude_version --no-preflight -- claude --version
kappaski eval benchmark --suite v0.10-claude-adapter-profile

Boundary

This version is an implemented local control-plane product surface with real binary validation.
