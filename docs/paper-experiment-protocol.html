Paper Experiment Protocol

This protocol defines how Kappaski turns public benchmark-inspired workloads into reproducible control-plane evidence.

Command

PYTHONPATH=src python -m kappaski.cli experiment paper-suite --out-dir .kappaski/paper-suite

Measurement Rule

External benchmark seeds measure agent workload credibility. Kappaski metrics measure control-plane behavior: observation, normalization, decision, mediation, enforcement claim, ledger persistence, proof, replay, evidence, and audit reconstruction.

Reporting Rule

Reports separate model task success from Kappaski control-plane success. Optional heavy external execution must be reported as skipped, unavailable, failed, or passed.
