v0.30 Experiment Case Runner

Status: implemented local experiment substrate. v0.30 converts benchmark-shaped simulated LLM-agent traces into Kappaski ledger, proof, replay, graph, evidence, metrics, and HTML reports. It does not claim official external benchmark execution.

Product Goal

Exercise the real Kappaski control plane from normalized cases while keeping the evidence level explicit.

Architecture

ExperimentSeed -> ExperimentCase -> simulated_agent_trace -> RuntimeEvent -> ledger/proof/replay/graph/evidence -> metrics/report

CLI

python -m kappaski.cli experiment list
python -m kappaski.cli experiment run --suite control-plane-core --out-dir .kappaski/experiments/core
python -m kappaski.cli experiment validate-fixtures --root benchmarks/experiments
python -m kappaski.cli experiment report --run .kappaski/experiments/core/run.json --out .kappaski/experiments/core/report.html

Claim Boundary

claim_scope=local_experiment_substrate, evidence_level=simulated_agent_trace, external_validation=not_run_optional.
