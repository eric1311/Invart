Core rule: v0.4 Skill/tool/capability scanner validation must use pinned snapshots
of real public Skill/MCP/tool corpora. Mock-only validation is not acceptable.

Implemented commands:

```bash
kappaski corpus scan --root benchmarks/corpora

kappaski eval benchmark --suite v0.4-real-skill-surface
```

## Implemented Closed Loop

- `kappaski corpus scan --root benchmarks/corpora` scans pinned real public snapshots.
- `kappaski daemon register-capabilities --target . --session <id> --corpus-root benchmarks/corpora --adapter <adapter>` records scanned surfaces as session capability grants.
- High-risk grant surfaces enter policy and create pending approvals.
- Proof export includes `capability_grants` and summary counts.
- `kappaski eval benchmark --suite v0.4-real-skill-surface` verifies scan, grant, approval, and proof behavior end to end.
