v0.17 complete

Transparent MCP Broker

MCP JSON-RPC messages can now be summarized and passed through without modification, creating a compatibility-first broker foundation.

What Shipped

Message Summary

src/kappaski/mcp_broker.py identifies tools/list, tools/call, tool name, message id, and folded raw content preview.

Transparent Step

transparent_broker_step returns the original JSON-RPC message unchanged plus Kappaski evidence.

CLI Smoke

mcp broker-step provides a deterministic way to validate broker evidence without needing a live MCP server.

CLI

kappaski mcp broker-step --message '{"jsonrpc":"2.0","id":1,"method":"tools/list"}'
kappaski mcp broker-step --message '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"write_file","arguments":{"path":".env","content":"SECRET=abc"}}}'

FieldMeaning
