v0.21 Path-Aware Policy

Deterministic policy over source, trust, taint, grant, resource, sink, and coverage facts.

Default Rules

secret read -> external network egress: deny
secret-tainted path -> CI/deploy mutation: require approval
external instruction -> destructive shell: deny

Deterministic critical policy cannot be downgraded by LLM review.

CLI

kappaski policy check-path --ledger ledger.jsonl --out path-policy.json
kappaski eval benchmark --suite v0.21-path-aware-policy

Metrics

The benchmark reports block rate, approval rate, and a benign false-positive proxy so safety and autonomy are measured together.
